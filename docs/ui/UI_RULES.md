# Layout Rules

## Cards

Avoid card-inside-card patterns.

Bad:

Card
Card
Card

Good:

Section
Divider
Content

## Borders

Use borders only when necessary.

Replace decorative borders with spacing.

## Typography

Page Title:
32px

Section Title:
20px

Card Title:
15px

Body:
14px

Metadata:
12px

## Tables

Prefer clean table layouts.

Avoid:

- boxed rows
- heavy borders

Prefer:

- subtle separators
- whitespace

## Colors

Green only for:

- positive ROI
- positive returns
- success states

Do not use green for general UI elements.

## Sidebar

Slightly darker than content area.

Must visually anchor the application.
