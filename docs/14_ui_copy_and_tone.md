# 14. UI Copy & Tone

How we speak to the user determines the level of trust they place in the platform.

## 1. The Voice: "The Senior Peer"
- **Professional but accessible.**
- **Analytical but empathetic.**
- **Neutral.** We don't say "Invest now!" we say "Based on these parameters, the result is..."

## 2. Key Terminology Guidelines

| Avoid | Use Instead | Why? |
| :--- | :--- | :--- |
| **Profit** | **Net Return** | "Profit" is vague; "Net Return" implies after taxes/fees. |
| **Loss** | **Drawdown** | In a market context, "Drawdown" is more technically accurate. |
| **Safe** | **Capital Protected** | "Safe" is a promise; "Capital Protected" describes a mechanic. |
| **Best** | **Optimized for X** | "Best" is subjective; "Optimized" implies a specific goal. |

## 3. Handling Numbers
- **Currency:** Always include the currency code (PLN, USD). Use thin-spaces for thousands (10 000 PLN).
- **Percentages:** Show to 2 decimal places (5.25%) unless it's a whole number (5%).
- **Abbreviations:** Explain abbreviations on first use or via tooltips (e.g., CPI - Consumer Price Index).

## 4. Success and Error Messaging
- **Success:** "Scenario calculated using current NBP rates."
- **Error:** "We couldn't find data for that bond issue. Please check the code."
- **Warning:** "This simulation assumes 10% inflation. If inflation drops, your returns will change."

## 5. Educational Microcopy
Short sentences placed near complex inputs:
- *Near Margin input:* "The margin is the fixed profit you earn above inflation."
- *Near Tax toggle:* "Belka tax (19%) is applied to your interest earned."

## 6. Call to Action (CTA)
Our CTAs are informative, not sales-driven:
- "Compare with S&P 500"
- "Add to my Notebook"
- "Download PDF Report"
- "Learn about EDO mechanics"
