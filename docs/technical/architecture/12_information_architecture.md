# 12. Information Architecture

The platform is structured to guide the user from simple curiosity to deep analysis.

## 1. Site Map

### A. Landing Zone
- **Home:** Featured calculators, latest bond rates, and "What is this?" intro.
- **Instruments Explorer:** A searchable catalog of Bonds, Stocks, Crypto, and Gold.

### B. The Calculator Suite (Verticals)
- **Bond Single-Calc:** Deep dive into one bond.
- **The Ladder Simulator:** For recurring monthly investments.
- **The "Emergency Fund" Check:** Comparing short-term bonds vs. cash.
- **Multi-Asset Comparison:** Side-by-side simulation.

### C. Planning Tools
- **Retirement Estimator:** Long-term wealth modeling.
- **Goal Planner:** Saving for specific targets (House, Education).

### D. Knowledge Hub
- **Education Center:** Guides, videos, and articles.
- **Glossary:** Definitions for "Capitalization," "CPI," "Coupon," etc.
- **Methodology:** Transparent explanation of our math and data sources.

### E. Personal Space
- **My Notebook:** The tracking dashboard.
- **Saved Scenarios:** Comparison history.

## 2. Navigation Strategy
- **Sticky Sidebar:** For quick access to different calculators.
- **Breadcrumbs:** To help users navigate deep instrument hierarchies.
- **Contextual Linking:** From a calculator result directly to a related educational article (e.g., "See how Belka Tax was calculated").

## 3. URL Structure (SEO Friendly)
- `/bonds/edo-10-year`
- `/calculate/bond-ladder`
- `/compare/edo-vs-sp500`
- `/education/how-inflation-linked-bonds-work`

## 4. Search Functionality
A global search bar (Cmd+K) that allows users to find:
- Specific bond issues (e.g., "EDO0834").
- Assets (e.g., "Bitcoin").
- Educational topics (e.g., "Tax").
- Their own saved scenarios.
